# Goal

建立 LinguaFlow 的桌面应用基础能力。

完成后，LinguaFlow 应具备桌面程序最基本的运行能力，包括：

- Application 生命周期管理
- MainWindow
- System Tray
- 最小化到托盘
- 托盘恢复主窗口
- 正常退出程序

本任务仍然不实现任何业务功能。

# 背景

Application 是整个程序唯一生命周期管理者。

Application 负责创建和管理：

- QApplication
- AppContext
- MainWindow
- SystemTray

除此之外，任何对象不得负责应用生命周期。

MainWindow 不知道 Tray 的存在。

Tray 不知道 MainWindow 的实现。

两者之间不得直接互相引用。

所有协调工作统一由 Application 完成。

---

# Scope

## 1. Application

新增：

src/app/application.py

职责：

- 创建 QApplication
- 创建 AppContext
- 创建 MainWindow
- 创建 Tray
- 初始化程序
- 启动事件循环
- 负责程序退出

Application 是整个程序唯一入口。

---

## 2. MainWindow

完善：

src/ui/main_window.py

要求：

- 空白窗口
- 设置窗口标题："LinguaFlow"
- 设置默认窗口尺寸
- 不添加菜单
- 不添加业务控件
- 不添加任何翻译相关内容

MainWindow 仅负责 UI。

不得管理任何业务对象。

---

## 3. Tray

新增：

src/ui/tray.py

要求：

支持：

- 显示托盘图标
- 双击恢复窗口
- 托盘菜单

菜单包含：

- 打开 LinguaFlow
- 退出

不要加入：

- 设置
- 历史记录
- 模型管理

---

## 4. 最小化行为

关闭窗口时：

不退出程序。

隐藏到托盘。

托盘菜单退出：

真正退出程序。

---

## 5. main.py

调整：

main.py 仅负责：

创建 Application。

调用：

Application.run()

不得包含其它业务逻辑。
