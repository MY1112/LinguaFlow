# Task ID

M1-001

---

# Title

项目初始化（Project Bootstrap）

---

# Milestone

Milestone 1 - Bootstrap

---

# 目标

完成 LinguaFlow 项目的基础初始化，使项目具备最小可运行能力。

本任务只负责项目基础设施，不实现任何业务功能。

---

# 前置要求

开始开发前必须阅读：

- AGENTS.md
- docs/architecture/00-开发原则.md
- docs/architecture/01-产品需求(PRD).md
- docs/architecture/02-工程架构.md
- docs/architecture/03-项目结构.md
- docs/architecture/04-开发规范.md

如发现文档冲突，立即停止并汇报。

---

# 本次任务

完成以下内容：

## 1、完善 pyproject.toml

要求：

- 使用 Python 3.12+
- 使用 pyproject.toml 管理依赖
- 添加项目基本信息
- 配置项目名称
- 配置运行依赖
- 配置开发依赖
- 配置 Ruff
- 配置 Black
- 配置 pytest

不要添加当前阶段不会使用的依赖。

---

## 2、完善 README.md

至少包含：

- 项目简介
- 技术栈
- 当前开发阶段
- 快速启动
- 项目目录
- 文档入口

---

## 3、补全 Python 包

所有源码目录必须成为 Python Package。

需要补充缺失的 __init__.py。

---

## 4、检查目录结构

确保目录符合：

docs/architecture/03-项目结构.md

如存在冲突，不修改架构。

---

## 5、建立程序入口

保证：

python -m src.main

能够正常启动。

允许：

打开一个空窗口。

不允许：

实现任何业务。

---

# 本次禁止开发

不要实现：

- 翻译
- llama.cpp
- OCR
- Tray
- Popup
- Hotkey
- Clipboard
- History
- Setting 页面
- Feature

---

# 完成标准（Definition of Done）

必须满足：

□ pyproject.toml 配置完成

□ README.md 完成

□ 所有 __init__.py 补齐

□ 项目可以启动

□ 空窗口能够正常显示

□ 程序能够正常退出

□ 无业务代码

□ 未修改项目架构

---

# 验收方式

执行：

python -m src.main

结果：

- 正常启动
- 出现空窗口
- 正常退出

视为完成。
