# Title

Foundation（基础设施）

---

# Milestone

Milestone 1

---

# Goal

建立整个项目最基础的公共基础设施。

本任务完成后，项目应具备统一的日志、配置、运行时上下文和应用启动能力，为后续所有 Feature 提供基础支撑。

本任务仍然不实现任何业务功能。

# Scope

## 1. pyproject.toml

完善项目配置：

- 项目名称：LinguaFlow
- Python >=3.12
- 配置运行依赖
- 配置开发依赖
- 配置 Ruff
- 配置 Black
- 配置 pytest

仅添加当前阶段需要的依赖。

---

## 2. README.md

完善：

- 项目介绍
- 技术栈
- 当前开发状态
- 快速启动
- 项目目录
- 文档入口

---

## 3. Python Package

补全所有缺失的：

__init__.py

确保所有源码目录均为 Python Package。

---

## 4. Logger

实现：

src/core/logger.py

要求：

提供统一 Logger。

整个项目以后只能使用这里提供的 Logger。

禁止：

print()

禁止：

各模块自行创建 Logger。

---

## 5. ConfigService

实现：

src/services/config_service.py

负责：

- 加载配置
- 保存配置
- 获取配置

不要直接读取 json。

统一通过 ConfigService。

---

## 6. AppContext

实现：

src/runtime/app_context.py

负责：

统一维护运行时对象。

允许维护：

- Logger
- ConfigService

不要提前加入：

- Model
- OCR
- Clipboard
- Hotkey

这些以后再加入。

---

## 7. MainWindow

创建：

空白 MainWindow。

要求：

能够正常启动。

能够正常关闭。

不要添加菜单。

不要添加业务 UI。

---
