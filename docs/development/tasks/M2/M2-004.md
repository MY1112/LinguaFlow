# M2-004 PromptService

## 目标

建立整个项目唯一的 Prompt 管理层。

PromptService 负责统一管理所有 Prompt 模板，并根据业务参数生成最终发送给模型的 Prompt。

它屏蔽 Prompt 的具体实现，使业务层无需关心 Prompt 内容。

---

## 职责

负责：

- 管理 Prompt 模板
- 根据参数生成 Prompt
- 提供统一 Prompt 接口
- 后续支持 Prompt 扩展

不负责：

- 模型加载
- 模型推理
- 翻译业务
- OCR
- UI
- 快捷键

---

## 目录位置

```
src/services/prompt_service.py
```

---

## 设计原则

PromptService 是整个项目唯一允许管理 Prompt 的地方。

禁止：

```
Feature
    ↓
自己拼接 Prompt
```

必须：

```
Feature
    ↓
PromptService
    ↓
ModelService
```

所有 Prompt 必须集中管理。

---

## 第一阶段支持

本阶段仅支持：

中译英

英译中

其它 Prompt 暂不实现。

---

## 对外接口

至少提供：

```python
build_translate_prompt(
    text: str,
    source_language: str,
    target_language: str,
) -> str
```

返回：

```
最终 Prompt 字符串
```

---

## Prompt 要求

Prompt 应尽量稳定。

例如：

```
你是一名专业翻译助手。

请将下面内容翻译为{目标语言}。

要求：

1. 保持原意。
2. 保持语气。
3. 不添加解释。
4. 仅输出翻译结果。

内容：

{text}
```

Prompt 必须集中维护。

禁止在 Feature 内再次拼接。

---

## 日志

统一使用 Logger。

记录：

- Prompt 构建
- Prompt 类型
- 异常

禁止 print()

---

## 异常

新增：

```
PromptServiceError
```

禁止向上暴露底层异常。

---

## Out Of Scope

本任务禁止实现：

- OCR Prompt
- 总结 Prompt
- 对话 Prompt
- UI
- 翻译 Feature
- 模型切换

---

- 能生成英译中 Prompt

- Ruff 通过
