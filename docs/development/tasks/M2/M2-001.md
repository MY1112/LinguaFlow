# Goal

建立 LinguaFlow 的 AI Runtime，为后续所有 AI 能力提供统一运行环境。

本任务只负责运行时框架，不加载模型，不调用 llama.cpp，不实现翻译。

# Scope

## 1、RuntimeState

新增：

src/runtime/runtime_state.py

负责维护整个 AI Runtime 的运行状态。

允许维护：

- initialized
- model_loaded
- current_model_path

不要加入任何业务状态。

---

## 2、RuntimeManager

新增：

src/runtime/runtime_manager.py

职责：

- 初始化 Runtime
- 关闭 Runtime
- 获取 RuntimeState

本阶段不要真正加载任何模型。

所有方法先完成框架。

---

## 3、Application

Application 创建 RuntimeManager。

程序退出时释放 Runtime。

不要加入任何 llama.cpp 代码。

---

## 4、AppContext

增加：

runtime

除此之外不要增加任何对象。

---

## 5、Logger

RuntimeManager 全部使用统一 Logger。

禁止使用 print()。

---
