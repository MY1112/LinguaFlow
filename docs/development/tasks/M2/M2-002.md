# Goal

接入 llama.cpp Python SDK，建立 LinguaFlow 唯一的模型适配层（Adapter）。

本任务目标是完成对第三方库的封装，使项目其它模块无需直接依赖 `llama_cpp`。

**本任务不实现翻译功能，不实现 Prompt，不实现 UI。**

# Scope

## 1、创建 Adapter

新增：

src/integrations/llama_cpp.py

职责：

封装 `llama_cpp` 第三方库。

Adapter 是项目中唯一允许 import `llama_cpp` 的地方。

其它模块禁止直接依赖第三方 SDK。

---

## 2、提供统一接口

实现以下能力：

- load_model(model_path)
- unload_model()
- is_loaded()
- generate(prompt, **kwargs)

说明：

- `generate()` 返回模型原始输出。
- 不负责 Prompt 拼接。
- 不负责翻译逻辑。
- 不负责结果解析。

---

## 3、异常处理

所有异常：

- 记录 Logger
- 抛出统一异常

禁止：

- print()
- 静默忽略异常

---

## 4、资源管理

要求：

重复加载模型时：

- 自动释放旧模型
- 再加载新模型

程序退出时：

- 能够安全释放模型资源

---

## 5、线程要求

本阶段不实现多线程。

所有接口保持同步调用。

不要提前加入：

- Thread
- QThread
- asyncio
- multiprocessing

---

# Out of Scope

- Popup
